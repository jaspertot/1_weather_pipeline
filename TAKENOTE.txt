For airflow, please setup:
 --> Install Ubuntu-24.04 WSL 
    wsl --install Ubuntu-24.04
 --> Install Python compatible for the WSL 
    sudo apt update
    sudo apt install python3 python3-pip python3-venv -y
--> Create new virtual environment compatible for WSL
    python3 -m venv .venv_wsl
--> Ensure that you put the newly created virtual environment to .gitignore
--> Activate the virtual environment
    source .venv_wsl/bin/activate
--> Install dependencies
    pip install -r .req/requirements.txt
--> For Airflow setup, run `airflow_setup.sh`
